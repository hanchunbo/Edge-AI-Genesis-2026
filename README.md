## 🗺 2026 核心路线图 (Roadmap)

### **Q1：工程基石与高性能体系 (Foundations)**
* **核心内容**：现代 C++ (11/14/17) 内存安全与性能进阶、Linux 高性能 I/O、CMake 大型工程构建、OpenCV 算子底层实现。
* **关键点**：RAII 机制、移动语义零拷贝、线程池架构、mmap 模型加载、像素级指针操作优化。

### **Q2：推理内功与数据分析模型 (Inference & Data)**
* **核心内容**：深度学习算子数学原理、ONNX 链路优化、ONNX Runtime C++ 集成、性能指标 P95/P99 建模分析。
* **关键点**：图优化理论、量化 (PTQ) 数学基础、数据驱动的推理延迟瓶颈调研、Qt UI 与推理逻辑异步解耦。

### **Q3：硬件加速双栈突击 (Acceleration)**
* **核心内容**：NVIDIA TensorRT GPU 加速、腾讯 NCNN 边缘端适配、YOLO 系列 (v8/v10) 全链路部署实战。
* **关键点**：INT8 量化校准、ARM NEON 指令集优化、高性能 NMS 算子手写、坐标框反算逻辑。

### **Q4：系统集成与职场升维 (System & Career)**
* **核心内容**：C++ AI 微服务 (Drogon)、Docker 容器化部署、云边协同逻辑实现、MEM 管理案例复盘。
* **关键点**：RESTful AI 接口封装、容器化环境压缩、关键路径 (CPM) 管理、硕士学位论文闭环。
